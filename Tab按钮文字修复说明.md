# 招投标合规审查系统 - Tab按钮文字颜色修复

## ✅ 本次更新

### Tab按钮文字颜色修复

**问题描述：**
- 原文预览和风险列表两个Tab按钮
- 点击其中一个后，另一个按钮的文字看不清楚
- 原因：未选中按钮是灰色背景，但文字是白色，对比度低

**修复方案：**
- ✅ 选中按钮：蓝色背景（#0078D4）+ 白色文字（#FFFFFF）
- ✅ 未选中按钮：灰色背景（#F0F0F0）+ 黑色文字（#000000）

**修改代码：**
```python
self.tab_view._segmented_button.configure(
    text_color="#000000",          # 未选中按钮文字颜色（黑色）
    selected_text_color="#FFFFFF"  # 选中按钮文字颜色（白色）
)
```

## 🎯 设计原则

**按钮文字对比度：**
- 高对比度 = 易读性
- 选中状态：蓝底白字（对比度高）
- 未选中状态：灰底黑字（对比度高）

**Windows 10/11设计规范：**
- 选中的Tab：主色调背景 + 白色文字
- 未选中的Tab：中性色背景 + 深色文字
- 确保所有状态下的文字都清晰可读
