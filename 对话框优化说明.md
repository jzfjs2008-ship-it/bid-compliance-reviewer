### 对话框布局优化

**之前的布局：**
```
┌─────────────────────┐
│         ⚠          │  ← 图标
│                     │
│   错误提示文字      │
│                     │
│      [确定]         │
└─────────────────────┘
```

**现在的布局：**
```
┌─────────────────────┐
│                     │
│   错误提示文字      │  ← 居中
│     (居中显示)      │
│                     │
│      [确定]         │  ← 标准按钮
└─────────────────────┘
```
