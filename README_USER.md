## 🚀 使用方法

双击桌面上的 `招投标合规审查系统.exe` 即可运行。

## 📋 功能说明

### 文件上传
- 支持拖拽或点击选择文件
- 支持格式：PDF、DOCX、XLSX
- 四个文件槽：招标文件、商务标、技术标、价格标

### 开始审查
1. 上传需要审查的文件
2. 点击"开始审查"按钮
3. 等待审查完成

### 查看结果
- **原文预览**: 查看上传文件的内容
- **风险列表**: 查看审查结果
  - 致命风险
  - 高危风险
  - 人工复核
  - 合规

### 导出报告
- 点击"导出报告"按钮
- 选择保存位置
- 生成PDF格式的审查报告

## ⚠️ 注意事项

1. 首次运行可能需要几秒启动时间
2. 程序为单实例，重复运行会激活已有窗口
3. 所有数据本地处理，不上传服务器
4. 仅提供辅助性形式审查，不构成法律意见
